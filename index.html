Console Project

Visual Studio Code is a Famouse Web Text Editor. This App is Made on Visual Studio Code. It is quite helpful.

Console is a section which isn't visible to people using the website unless and until they *Inspect. It is used to check if the output is correct or not. To open console screen you need to right click in the website area and then click on Inspect and select console up. You need to edit the code to add text in the console.
